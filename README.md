# 🌱 Prosper

> **Prosper — Free and open-source personal finance software for everyone.**

Prosper is a modern, community-driven personal finance platform that empowers individuals to take control of their money, build healthy financial habits, and achieve financial well-being—without subscriptions, advertisements, or vendor lock-in.

Whether you're a student living away from home, a young professional managing your first salary, or a family tracking household expenses, Prosper is designed to make personal finance simple, secure, and accessible to everyone.

---

## ✨ Features

* 💰 Expense and income tracking
* 📊 Budget management
* 🏷️ Expense categories and tags
* 📈 Financial reports and analytics
* 🔄 Recurring transactions
* 🎯 Savings goals and budgeting tools
* 📱 Responsive design for desktop and mobile
* 🔒 Secure and privacy-focused
* 🆓 Completely free and open source

## 🚀 Why Prosper?

Many personal finance applications lock essential features behind subscriptions or require users to share sensitive financial information with third parties.

Prosper takes a different approach:

✅ No subscription fees
✅ No advertisements
✅ No vendor lock-in
✅ Self-host so you have complete control
✅ Built by the community, for the community

## 📦 Installation

### Requirements

* PHP 8.1+
* MySQL 5.7+ or MariaDB 10.4+
* Apache or Nginx

### Quick Start

```bash
git clone https://github.com/prosperaccounting/prosper.git
cd prosper
```

1. Create a database.
2. Navigate to your folder
3. Open Prosper in your browser.
4. Fill Up essential information on install page.
5. Start Accounting
